# 🎯 爆旋陀螺X 收藏工具

一個完全免費、开源嘅爆旋陀螺X收藏追蹤工具。任何人都可以自己使用，資料完全存在你本地。

## ✨ 功能特色

- 📦 **收藏管理** - 記錄你擁有的陀螺，支援評分同備註
- 📚 **全球圖鑑** - 50+款陀螺完整資料
- 🏆 **冠軍組合** - 專業玩家驗證的最強配搭
- 🔍 **智能搜尋** - 快速找到想要的陀螺
- 📱 **手機友好** - 可安裝為PWA，離線使用
- 💾 **本地儲存** - 資料永遠存在你的瀏覽器
- 🔄 **自動更新** - 每日獲取最新陀螺資訊

## 🚀 立即使用

### 方式一：線上使用（推薦）
直接打開以下連結：
```
https://sunnycsl.github.io/beyblade-x-tracker/
```

### 方式二：本地使用
1. 下載此倉庫
2. 解壓後雙擊 `index.html`
3. 完全離線可用

## 📱 安裝為手機App

1. 用手機瀏覽器打開上方連結
2. iPhone：點擊分享按鈕 → 「加入主畫面」
3. Android：點擊右上角選單 → 「安裝應用程式」

## 💾 資料備份

你的資料只會存在瀏覽器的 localStorage。如需備份：

1. 進入「工具」分頁
2. 點擊「導出資料」
3. 會下載一個 JSON 檔案

如需恢復：進入「工具」分頁 → 點擊「導入資料」

## 🔧 技術架構

- **前端**: 純 HTML/CSS/JavaScript，無需任何框架
- **數據**: JSON 格式，全球數據存放於 GitHub
- **緩存**: Service Worker，支援離線使用
- **存儲**: 瀏覽器 localStorage
- **更新**: 每日自動从 GitHub fetch 最新數據

## 📁 檔案結構

```
beyblade-x-public/
├── index.html        # 主頁
├── manifest.json     # PWA 配置
├── sw.js            # Service Worker
├── data/
│   ├── global_db.json   # 全球陀螺數據
│   ├── combos.json      # 冠軍組合
│   └── parts.json       # 零件配搭建議
└── README.md
```
